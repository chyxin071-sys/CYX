# 品诺筑家 - 高端私宅定制提案工作台 (Zhujia_Pre)

这是一个专为室内设计师和家装行业打造的**纯本地、轻量级、交互式**提案演示工具。
它旨在替代传统臃肿的 PPT，让设计师能够以“高级杂志画报”的形式，更优雅、更直观地向业主展示设计方案。

## 🌟 核心特性

- **纯本地运行**：无需注册，无需登录，所有数据安全地保存在本地浏览器中，随时随地可用（支持离线演示）。
- **多方案管理**：支持一键创建多个客户方案，分别管理不同业主的提案数据，互不干扰。
- **多空间导航**：内置“客厅、餐厅、厨房、主卧”等常用空间，支持自由拖拽排序和增删。
- **多图轮播与精准标注**：右侧支持上传多张高清效果图，点击图片即可添加“带呼吸灯动画”的细节引线标注。标注与具体图片绑定，切换图片自动隐藏。
- **卡片化模块编辑**：左侧预设 4 大专业讲解模块（情绪风格、色彩材质、灯光氛围、设计细节）。支持悬浮卡片式交互、拖拽排序、显隐控制。
- **沉浸式演示模式**：一键切换“工作台”与“沉浸提案”模式。提案模式下会自动隐藏所有编辑按钮、上传占位符和无关 UI，只保留最纯粹的设计画面，提升见客逼格。
- **原生色彩与材质库**：支持点击色块呼出系统调色盘提取主色调，支持本地上传核心材质实拍图（如岩板、木饰面）。

## 🚀 快速开始

因为是纯前端应用，不需要配置任何复杂的环境：

1. 克隆本项目到本地：
   ```bash
   git clone https://github.com/chyxin071-sys/Zhujia_Pre.git
   ```
2. 双击打开根目录下的 `index.html`，即可直接在浏览器中使用。

## 🛠️ 技术栈

- **HTML5 + Vanilla JavaScript**：零框架依赖，极致轻量，加载飞快。
- **Tailwind CSS**：通过 CDN 引入，实现现代化、响应式的高级 UI 排版。
- **SortableJS**：实现模块和空间 Tab 的丝滑拖拽排序。
- **LocalForage / IndexedDB (计划中)**：用于大容量本地数据持久化存储。

## 👨‍🎨 适用场景

- 室内设计师向客户进行方案汇报
- 软装设计师进行色彩与材质搭配展示
- 装修公司标准化谈单流程工具
