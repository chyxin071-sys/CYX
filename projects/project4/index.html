Project 04 / Lifestyle App / Digital Museum

个人生活轨迹管家

一款以“每日一图一文”为核心，融合空间轨迹分析的极简私密生活记录工具。

01 / Positioning & Goals

产品目标

帮助用户在快节奏生活中捕捉瞬时幸福，通过时间与空间双维度的可视化，构建属于个人的“数字人生博物馆”。

核心价值

  • 极致丝滑的视觉体验： 对视觉审美有高要求，喜欢精致、高级、有呼吸感的 UI 风格。
  • 去社交化的纯净记录： 仅为自己记录生活碎片，不承受复杂手帐或社交平台的压力。
  • 高度自动化的时空叙事： 直观看到自己人生轨迹的变迁。

02 / Core Features

1. 智能记录系统
  • 快速入口：显著的打卡入口，支持一键调用相机或相册。
  • 元数据抓取：自动读取照片 EXIF 信息（时间、地点、经纬度）。
  • 补打卡：支持历史日期的回溯记录。
2. 多维日历系统
  • 四级缩放：日、周、月、年（3x4 布局）四种视图无缝切换。
  • 视觉留白：未打卡日期显示为纯净留白，无负向压力。
3. 空间分析系统
  • 足迹地图：极简色调底图，展示地理坐标点。
  • 时序连线：按照记录先后顺序绘制淡色虚线轨迹。
  • 热力渐变：根据在各省份/城市停留的时间长度，呈现由浅至深的区域渐变色。
4. 导出与总结
  • 多模版导出：日（拍立得）、周（长图）、月（日历图）、年（全景年历）。
  • 可视化报表：独立总结页，展示空间分布、活跃度曲线与年度色谱。

03 / UI & UX Guidelines

设计语言： 极简主义（Minimalism），大面积留白，动态取色。

交互表现：
- 照片详情页背景色随照片主色调实时渐变（毛玻璃效果），呈现沉浸感。
- 打卡成功触发粒子汇聚成星动效，伴随轻微震动反馈（Haptic Feedback）。
- 模仿 iOS 相册的视图缩放体验，平滑无缝。
